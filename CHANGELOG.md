* 2020/2/14 ATM3.0構思 : 存取帳號的方法
    * 想法:
        * 多個矩陣 (帳號/密碼/金錢) / 二維矩陣
        * type acc interface{
            login() -> 登入func
        }
    * 流程
        1. 輸入帳號
        2. for尋找該帳號的索引數 -> 存入a變數
        3. 輸入密碼
        4. if 判斷該密碼是否與密碼矩陣的該位置相符(pass[a])
            * true
                * 登入成功
                * a變數存入此次登入變數
            * false
                * 登入失敗
                * a變數歸零
                * 回到1.
