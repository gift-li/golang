### 整體規劃
* 開始/啟動程式 -> 系統(登入) -> 離開/結束程式
------
### 系統規劃
* 登入 -> 清單 -> 登出  (強制)
    ^-------------|
    * 登入
        * -> 新增帳號
        * -> 登入ATM系統 (若無帳號無法輸入 -> 登入)
        * -> 離開系統
    * 清單
        1. 個人資料檢視
        2. 修改資料
            * -> 修改帳號名稱
            * -> 修改密碼
        3. 金流
            * -> 領錢
            * -> 存錢
        4. 登出
            * -> 確定要登出? (二次驗證)
    * 登出
        * -> 登入
